🏨 Hotel Reservation Prediction

Predict whether a hotel reservation is likely to be Confirmed or Cancelled using Machine Learning.

📖 About the Project

Hotel reservation cancellations can significantly impact hotel operations and revenue. This project leverages Machine Learning to analyze reservation details and predict whether a booking is likely to be confirmed or cancelled before the guest arrives.

  • 👨 Number of Adults
  • 👶 Number of Children

  • 🗓️ Arrival Month
